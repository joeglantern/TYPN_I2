# TYPNI - The Young People's Network International

A comprehensive youth empowerment platform built with Next.js, Three.js, and Supabase.

## 🌟 User Features

### Authentication & Profile
- **User Registration & Login**
  - Email-based authentication
  - Social media login options
  - Password recovery system
  - Email verification

- **Profile Management**
  - Profile picture upload
  - Personal information updates
  - Privacy settings
  - Activity history

### Main Features

#### 📚 Blog
- Read community stories and updates
- AI-powered blog reading assistant
- Comment on articles
- Share posts on social media
- Save articles for later reading

#### 👥 Programs
- Browse available youth programs
- Apply for programs
- Track application status
- Access program resources
- Program completion certificates

#### 📅 Events
- View upcoming events calendar
- Register for events
- Event reminders
- Virtual event access
- Event photo galleries

#### 🖼️ Gallery
- Browse community photos
- View event highlights
- Submit your own photos
- Create photo collections
- Image download options

#### 💬 Community
- Real-time chat system
- Discussion forums
- Create and participate in polls
- Member directory
- Direct messaging

#### 🤝 Membership
- Apply for TYPNI membership
- Track application progress
- Access member-only content
- Network with other members
- Member benefits dashboard

#### ❤️ Donations
- Make secure donations
- Choose donation frequency
- Track donation history
- View impact reports
- Tax receipt generation
